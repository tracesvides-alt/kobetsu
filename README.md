# 米国株 個別銘柄チェッカー (US Stock Analysis Dashboard)

米国株の個別銘柄を、ファンダメンタルズとテクニカルの両面から深く分析するためのダッシュボードです。

## 📌 機能の概要

### 1. 企業プロファイルと業績可視化
*   **基本情報の表示:** 時価総額、PER、PBR、配当利回り、52週レンジといった基本的な株価・指標データを一目で確認できます。
*   **事業概要の自動生成:** AIが企業のビジネスモデルや主力事業について簡潔に要約し、その企業が「何で稼いでいるのか」を瞬時に把握できます。
*   **業績推移チャート:** 「年次」と「四半期」を切り替えて表示できるチャート機能。売上高と純利益の推移、および営業キャッシュフローの推移を視覚的に捉え、企業の成長トレンドを確認できます。

### 2. 財務の質とバリュエーション (Quality & Value)
*   **ピオトロスキー F-Score 診断:** 収益性、流動性/財務レバレッジ、業務効率性の3分野・全9項目から企業の財務健全性を自動評価。「なぜクリアしたのか」「なぜダメだったのか」という具体的な理由付き（例: 「流動比率が改善し支払い能力が向上」等）で判定結果を表示します。
*   **DCF方式による理論株価計算:** 過去のフリーキャッシュフロー実績を基に、スライダーを使って「想定成長率」と「割引率(WACC)」を自由に調整し、現在の株価が割安か割高かをシミュレーション計算できます。

### 3. タイミングと戦略 (Timing & Strategy)
*   **インタラクティブ・チャート:** 日足のローソク足チャートに、SMA（20日/50日/200日移動平均線）、RSI（相対力指数）、MACDを重ね合わせて表示。さらに短期・中期・長期のモメンタムトレンドを矢印アイコンで直感的に可視化します。
*   **ミネルヴィニの「成長株トレンド」判定:** 成長株投資の巨匠マーク・ミネルヴィニの「トレンドテンプレート（Stage 2）」の基準をプログラムで再現。移動平均線の関係性や52週高値圏との距離など、8つの厳格な条件をどれだけ満たしているかを自動判定します。
*   **需給動向とインサイダー分析:** 機関投資家の保有比率、インサイダー（経営陣など）による直近の自社株買い・売り動向、および市場の空売り比率を表示し、大口投資家の動きを把握します。

### 4. ピアグループ（競合）比較マトリックス
*   **比較対象の自動抽出:** 分析中の銘柄に関連する競合他社を独自に抽出。
*   **相対ランキング機能:** 売上成長率、営業利益率、PER、配当利回りの4つの指標で競合企業群をランク付けし、「総合順位」を算出。どの企業が業界内で最も優位性を持っているかを表形式で可視化します。（全指標の1位や、ターゲット企業の順位も強調表示されます）。

### 5. プロ目線の総合投資判断 (CIOダッシュボード)
*   **5軸レーダーチャート:** 「成長性」「トレンド」「安全性」「割安性」「モメンタム」の5つの軸で銘柄の総合力をグラフ化。
*   **VCP（ボラティリティ収縮パターン）解析:** 直近の価格推移からボラティリティが収縮している兆候（エントリーのタイミングが近いか）を自動分析します。
*   **自動トレードプラン算出:** テクニカルな高値を基準とした「買いのブレイクアウトポイント」、リスク管理のための「絶対損切りライン」、および推奨される「安全なポジションサイズ（資金の何%までか）」を具体的な数値で提示します。

### 6. AIアナリストレポート (要設定)
*   **強みと弱みの深掘り:** AIが現在の相場環境に基づいたポジティブ要素（強み）とネガティブ要素（リスク）をリストアップ。
*   **将来見通しの考察:** 今後の業績ドライバーとなる要因や懸念事項をレポート形式で簡潔に出力し、最終的な投資判断のブレインストーミングをサポートします。
